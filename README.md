# Janrenzo Facto — Freelance Portfolio Website

A modern, premium, and conversion-focused portfolio website for **Janrenzo Facto**, a freelance WordPress Website Designer, Front-End Developer, GoHighLevel Specialist, and Graphic/Web Creative.

This portfolio was built to showcase my skills, selected works, professional experience, services, tools, and contact information in a polished and client-ready presentation.

🔗 **Live Website:** https://janrenz.netlify.app/  
📁 **Repository:** https://github.com/FactoJanrenzo/newjanrenzo-portfolio

---

## Overview

This website is designed as a personal freelance portfolio for attracting clients worldwide. It highlights my experience in building high-converting websites, landing pages, GoHighLevel funnels, SEO-friendly page structures, and responsive digital experiences.

The design direction focuses on:

- Premium dark theme with lime accent colors
- Smooth animations and modern interaction effects
- Mobile-responsive layout
- Portfolio showcase sections
- Contact form powered by Netlify Forms
- Thank You page after form submission
- Clean, fast, and deployment-ready Vite + React setup
